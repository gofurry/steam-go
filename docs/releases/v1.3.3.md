# v1.3.3 Release Notes

v1.3.3 is a small compatible feature release focused on bounded read-only Web
helpers and a thin VDF addon bridge.

## Highlights

- Added `addons/vdf`, a thin bridge to `github.com/gofurry/vdf-go v1.0.0`.
- Added a bounded Storefront reviews collector for small in-memory review reads.
- Added a local-only Community inventory helper that pairs assets with their
  matching descriptions.

## Added

- Added `addons/vdf` for text Valve Data Format / KeyValues parsing,
  querying, editing, and marshaling through `vdf-go`.
- Added `examples/vdf` and bridge tests for the VDF addon.
- Added `client.Web.Storefront.CollectAppReviews`.
- Added `storefront.CollectAppReviewsOptions` and `storefront.AppReviewsCollection`.
- Added `community.JoinInventoryDescriptions`.
- Added `community.JoinedInventoryItem`.

## Notes

- `CollectAppReviews` requires `MaxPages` or `MaxReviews`; use
  `ListAppReviews` for streaming page-by-page processing without retaining all
  reviews in memory.
- `JoinInventoryDescriptions` is local-only. It does not fetch prices, inspect
  market state, trade, sell, or automate account behavior.
- `addons/vdf` does not implement binary VDF, parse `shortcuts.vdf`, scan Steam
  installations automatically, or read local user directories unless the caller
  explicitly passes a path.
- This release does not add account automation, purchase, trade, sell, browser
  fallback, or upstream limit bypass behavior.

## Compatibility

- No breaking changes.
- New APIs are additive.
- `github.com/gofurry/vdf-go v1.0.0` is added as a direct dependency.

## Validation

- `go test ./...`
- `go vet ./...`
- `go run ./examples/vdf -file .\testdata\fixtures\addons\vdf\appmanifest_730.acf -key AppState`
