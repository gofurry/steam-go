# steam-go v1.0.0

`steam-go v1.0.0` 是项目的首个正式稳定版。

它定位为一个面向官方 Steam Web API 的稳定 Go SDK，并提供面向生产使用的请求控制能力和清晰的发布边界。

## 重点

- 稳定的 root `Client`，通过 `client.API.*` 访问服务分组
- 当前公开服务面下的 typed Steam Web API 支持
- 基于 functional options 的 API key、access token、timeout、retry、rate limit、proxy selection 和 traffic policy 配置
- 面向生产使用的请求控制能力：
  - `WithSafeDefaults()`
  - retry / backoff 控制
  - rate limiting
  - host / session request controls
  - sticky proxy selection
  - health-checked proxy pools
  - proxy metrics snapshots
- 官方 Web API 流量与未来公开商店页基础能力之间的 traffic-class 隔离
- 稳定的 addon 入口：
  - `addons/openid`
  - `addons/a2s`

## 文档与发布准备

- 已发布兼容性策略
- 已发布 endpoint 稳定性说明
- 已发布 endpoint 覆盖范围说明
- 已整理 `v1.0.0` release notes
- 真实 live smoke 验证已统一到 `examples/live/`

## 项目边界

`v1.0.0` 不承诺：

- 完整 Steam Store SDK
- 完整 Steam Community SDK
- 内置浏览器 fallback 实现
- 未记录页面 payload 形状的长期稳定性
- 超出当前文档范围的新 API 覆盖

## 验证

`v1.0.0` 发布线基于以下验证：

- `go test ./...`
- `go test -race ./...`
- `go vet ./...`
- `staticcheck`
- `govulncheck`

## 备注

- `TrafficClassPublicStorePage` 仍然是策略与基础设施能力，不代表 `v1.0.0` 已经提供完整公开 Steam 商店页抓取 API。
- 官方 Steam Web API 的后续扩展预计在 `v1.0.0` 之后再恢复。
