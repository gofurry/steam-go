# v1.3.2 Release Notes

v1.3.2 是一个兼容性小版本，重点补齐 Steam Store 新闻 / 活动采集，以及下游内容清洗能力。

## 亮点

- Storefront 新增 Steam event/news 页面使用的公开 adjacent partner events 接口。
- 新增 `addons/markup`，用于转换常见 Steam BBCode、清洗 HTML，并生成适合索引或 metadata 的纯文本摘要。

## 新增

- 新增 `client.Web.Storefront.GetAdjacentPartnerEvents` 与 `GetAdjacentPartnerEventsRaw`，用于访问 `store.steampowered.com/events/ajaxgetadjacentpartnerevents`。
- 新增 Store events typed response model，并为波动的嵌套字段保留 raw payload。
- 新增 `addons/markup` helpers：
  - `SteamBBCodeToHTML`
  - `CleanSteamContent`
  - `CleanHTML`
  - `PlainText`
  - `Summary`
- 支持常见 Steam 内容格式，包括 `{STEAM_CLAN_IMAGE}`、`[/*]` 列表结束符、转义括号文本和转义 URL 属性。
- 新增 `examples/markup` 可运行示例。

## 变更

- Golden tests 在比较前归一化换行，避免 Windows / Unix checkout 差异造成测试假失败。
- 英文和中文文档补充 Store events 与 markup helper reference。

## 说明

- Store events 仍属于非官方 Web surface。SDK 只暴露稳定 typed 子集，并为波动字段保留嵌套 raw payload。
- Markup helper 默认执行安全清洗，适合接入调用方自己的存储、索引或渲染流水线。
- YouTube iframe 默认不输出；调用方可通过 `markup.WithYouTubeEmbeds(true)` 显式开启。
- 本版本不新增账号自动化、购买、交易、出售或 browser fallback 行为。

## 兼容性

- 无 breaking change。
- 新 API 均为兼容性加法。
- 新增 `github.com/microcosm-cc/bluemonday` 作为 HTML sanitize 的直接依赖。

## 验证

- `go test ./...`
- `go run ./examples/markup`
