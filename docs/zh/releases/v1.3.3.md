# v1.3.3 Release Notes

v1.3.3 是一个兼容性小版本，重点补齐有界只读 Web helper，并新增轻量 VDF addon
桥接。

## 亮点

- 新增 `addons/vdf`，轻量桥接 `github.com/gofurry/vdf-go v1.0.0`。
- 新增有界 Storefront reviews collector，适合小规模内存聚合。
- 新增纯本地 Community inventory helper，用于把 assets 与匹配的 descriptions
  配对。

## 新增

- 新增 `addons/vdf`，通过 `vdf-go` 支持文本 Valve Data Format / KeyValues
  解析、查询、编辑和 marshal。
- 新增 `examples/vdf` 和 VDF bridge tests。
- 新增 `client.Web.Storefront.CollectAppReviews`。
- 新增 `storefront.CollectAppReviewsOptions` 和 `storefront.AppReviewsCollection`。
- 新增 `community.JoinInventoryDescriptions`。
- 新增 `community.JoinedInventoryItem`。

## 说明

- `CollectAppReviews` 必须设置 `MaxPages` 或 `MaxReviews`；如果希望逐页处理且
  不把 reviews 留在内存中，继续使用 `ListAppReviews`。
- `JoinInventoryDescriptions` 是纯本地 helper。它不获取价格、不判断市场状态、
  不交易、不出售，也不做账号自动化。
- `addons/vdf` 不实现 binary VDF，不解析 `shortcuts.vdf`，不自动扫描 Steam
  安装目录，也不会在调用方未显式传入路径时读取本机用户目录。
- 本版本不新增账号自动化、购买、交易、出售、browser fallback 或绕过上游限制的能力。

## 兼容性

- 无 breaking change。
- 新 API 均为兼容性加法。
- 新增 `github.com/gofurry/vdf-go v1.0.0` 作为直接依赖。

## 验证

- `go test ./...`
- `go vet ./...`
- `go run ./examples/vdf -file .\testdata\fixtures\addons\vdf\appmanifest_730.acf -key AppState`
