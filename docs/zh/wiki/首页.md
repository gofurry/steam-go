# steam-go Wiki

### [English Version](https://github.com/GoFurry/steam-go/wiki)

欢迎来到 `steam-go` Wiki。

`steam-go` 是一个轻量级的 Go 语言 Steam Web API SDK，同时也提供了一组面向真实使用场景的工具能力，包括客户端配置、凭证管理、重试与限流、代理选择、流量策略、OpenID 登录验证、A2S 服务器查询、错误处理以及安全 URL 脱敏。

这个 Wiki 的目标是帮助开发者快速理解 SDK 的组织方式、各个工具功能的用途，以及如何通过简短、实用的示例使用它们。
